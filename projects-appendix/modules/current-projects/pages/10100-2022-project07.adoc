== Questions
Linking data together in 'R' can be done horizontally and vertically. A [.underline]*#horizontal#* merge combines dataframes horizontally which means that it adds variables (aka columns) to an exisiting data frame according to a common shared ID field. +
A [.underline]*#vertical#* merges the data.frames ontop of eachother. The data.frames have the same variables but different observations (aka rows). +
